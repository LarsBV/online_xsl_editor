Welcome

This is a hopefully simple xslt editor. It has been tested in chromium and firefox.

If you move the mouse over the gray strip at the top of a window you'll get some more buttons, for uploading, saving etc.., and you may move the windows.

Any changes to content are automatically saved, window positions are not yet saved.

It uses your browsers internal xslt processor so result might vary a bit between browsers.

The editor component is a ACE editor, shortcuts can be found here

Useful shortcuts are among others, Ctrl+Space, autocompletion. and Ctrl+H, regex search.

There is room for many improvements, and it was just to be a quick tool for another project. So please join me at github, the source code license is simplified bsd.

Any comments, or just a thank you will be much appreciated.
